How to read this: These numbers show how many times you could send that entire book as input to each model for exactly one dollar. This is a pure ingestion cost — not generation. If you asked the model to summarize, analyze, or chat about the book, output tokens would add extra cost (typically 3–10× more expensive per token).

The open-source caveat: Llama, Mistral, and DeepSeek prices assume hosted inference (Together.ai / Fireworks). Run them locally on your own hardware and the cost approaches $0.
